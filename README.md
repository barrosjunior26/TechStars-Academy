# TechStars Academy

TechStars Academy é um projeto acadêmico desenvolvido em ASP.NETCore MVC. Este site escolar oferece uma plataforma abrangente para a gestão de alunos, lançamento de notas, publicação de desempenho, cadastro de gestores, divulgação de carreiras disponíveis na instituição e uma página de novidades.



1. Funcionalidades
   Cadastro de Alunos: Permite o registro de novos alunos na instituição.

2. Lançamento de Notas: Facilita o lançamento de notas das disciplinas escolares.

3. Publicação de Desempenho: Exibe o desempenho dos alunos de forma clara e organizada.

4. Cadastro de Gestores: Permite o registro de gestores da instituição.

5. Divulgação de Carreiras: Publica as carreiras disponíveis na instituição.

6. Página de Novidades: Mantém os alunos e gestores informados sobre as últimas novidades e eventos.
   
   

## Tecnologias Utilizadas

- ASP.NETCore MVC: Framework para desenvolvimento web.

- Entity Framework Core: ORM para acesso a dados.

- Bootstrap: Framework CSS para design responsivo.

- JavaScript: Para interatividade e funcionalidades dinâmicas.



### Como Executar o Projeto



1. Clone o repositório:
   git clone https://github.com/barrosjunior26/TechStars-Academy.git

2. Navegue até o diretório do projeto:
   cd techstars-academy

3. Restaure as dependências:
   dotnet restore

4. Execute o projeto:
   dotnet run
